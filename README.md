# User Data Table

A simple JavaScript project that fetches user data from the JSONPlaceholder API and displays it in a responsive Bootstrap table.

## 📸 Preview

Displays user information including:

- Serial Number
- Name
- Email
- Website
- Action Buttons (Edit & Delete)

## 🛠️ Technologies Used

- HTML5
- Bootstrap 5
- JavaScript (ES6)
- Fetch API
- JSONPlaceholder API

## 📂 Project Structure

```
user-data-table/
│
├── index.html
```

## ⚡ Features

- Fetch data from external API
- Dynamically render user data
- Responsive Bootstrap table
- Clean and simple UI
- Action buttons for future CRUD functionality

## 🌐 API Used

JSONPlaceholder Users API

https://jsonplaceholder.typicode.com/users

## 📖 How It Works

1. JavaScript sends a request using the Fetch API.
2. Data is received from JSONPlaceholder.
3. User information is dynamically inserted into the table.
4. Bootstrap styles the table and buttons.

## 🔧 Installation

Clone the repository:

```bash
git clone https://github.com/asadujjaman44-tech/user-data-table.git
```

Open the project folder:

```bash
cd user-data-table
```

Run `index.html` in your browser.
